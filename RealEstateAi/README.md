# Urban Living Realty

A modern real estate website built with HTML, CSS, PHP, JavaScript, and MySQL.

## Features
- Responsive landing page
- Property listing pages
- Property detail page
- Contact form with database-ready submission
- MySQL schema included

## Run locally
1. Start Apache and MySQL in XAMPP/WAMP.
2. Import the SQL from schema.sql into your MySQL server.
3. Place this project inside your web server root.
4. Open http://localhost/RealEstateAi/index.php

## Notes
- The PHP config uses the default local MySQL credentials: root / empty password.
- If your environment uses different credentials, update config/database.php.
